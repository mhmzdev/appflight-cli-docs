## Setup

1. **Generate an API key** — AppFlight mobile app → Settings → API Keys → **+** — label it `Bitrise`
2. **Add secret** — Bitrise → Secrets → Add new → `APPFLIGHT_API_KEY` (mark as protected)
3. **Commit `appflight.json`** to your repository

## bitrise.yml

Add a Script step after your Android Build step:

```yaml
- script@1:
    title: Install AppFlight CLI
    inputs:
      - content: dart pub global activate appflight_cli

- script@1:
    title: Upload to AppFlight
    inputs:
      - content: |
          #!/usr/bin/env bash
          set -ex
          appflight_cli upload --flavor stage --ci
    envs:
      - APPFLIGHT_API_KEY: $APPFLIGHT_API_KEY
```

## Multi-flavor

```yaml
- script@1:
    title: Upload stage to AppFlight
    inputs:
      - content: appflight_cli upload --flavor stage --ci
    envs:
      - APPFLIGHT_API_KEY: $APPFLIGHT_API_KEY

- script@1:
    title: Upload qa to AppFlight
    inputs:
      - content: appflight_cli upload --flavor qa --ci
    envs:
      - APPFLIGHT_API_KEY: $APPFLIGHT_API_KEY
```

> **Note:** Install the CLI once at the start of the workflow and reuse it across steps.
