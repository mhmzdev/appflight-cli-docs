The AppFlight mobile app is the companion to the CLI. After you upload an APK, testers and admins interact with the build through the app — reviewing details, accepting invites, and receiving push notifications.

## Download

<a href="https://play.google.com/store/apps/details?id=dev.mhmz.app_flight" target="_blank">
  **Get it on Google Play →**
</a>

Available on Android.

## What the app does

| Capability | Description |
|---|---|
| Manage apps | Create apps, invite testers, generate API keys |
| Review builds | See APK metadata, version history, release notes |
| Tester notifications | Receive push notifications when a new build is uploaded |
| Subscription | Upgrade to First Class from **Settings → Subscription** |

## Why you need it

The CLI handles uploads. The app handles everything else:

- **API keys** — generated inside the app under **Settings → API Keys**; required for `appflight login --api-key`
- **Tester management** — invite testers and manage access per app
- **Subscription** — upgrade or check your current plan

## Requirements

- Android 8.0 (API 26) or higher
- An AppFlight account (sign up free inside the app)
